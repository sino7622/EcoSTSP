# EcoSTSP - 南科生態地圖查詢平台 🦉

EcoSTSP 是一個基於 Leaflet.js 與 Esri Leaflet 開發的互動式地理資訊平台，旨在展示與監測南台灣科學園區（南科）各園區的生物多樣性與生態現況。



## 🌟 主要功能

* **多底圖切換**：支援 Esri 高解析度衛星影像及科技感十足的深灰色底圖切換。
* **動態生態標點**：核心園區（沙崙）具備動態漣漪效果，強調監測重點。
* **多物種展示**：彈窗支援橫向並列的正方形（1:1）物種照片展示。
* **智慧縮放導覽**：側邊欄連動地圖縮放，自動切換全域文字標籤與詳細資訊點位。
* **園區範圍視覺化**：整合 KML 資料，清晰呈現各園區邊界範圍。

## 📸 監測園區與物種

本平台涵蓋南科轄下多個園區，包含：
- **沙崙園區**：草鴞
- **嘉義園區**：黑翅鳶、環頸雉
- **台南園區**：環頸雉、燕鴞
- **高雄/橋頭園區**：草鴞、環頸雉、燕鴞
- **屏東園區**：燕鴞、小雲雀
- **楠梓園區**：(資料建置中)

## 🛠️ 技術棧

- **前端框架**: [Tailwind CSS](https://tailwindcss.com/)
- **地圖引擎**: [Leaflet.js](https://leafletjs.com/)
- **GIS 資料服務**: [Esri Leaflet](https://esri.github.io/esri-leaflet/)
- **資料格式**: KML (透過 leaflet-omnivore 解析)
